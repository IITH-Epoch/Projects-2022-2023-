# Style-Transfer

This project was developed by [Divyanshu Bhatt](https://github.com/Divyanshu-Bhatt), Club Member of [Epoch](https://github.com/IITH-Epoch) 2022-23

The project replicates the research paper ["A Neural Algorithm of Artistic Style"](https://arxiv.org/abs/1508.06576). Given the content and style image, the model tries to transfer the style to the content image using a pre-trained VGG-19 network. 

**Pros:** The user gets to choose both the content and the style image. **Cons:** It is a computationally expensive process.

**Possible Improvement:** Using a similar method as above, one can train a model on specific style images and the image can be generated by just a single pass through the network.
